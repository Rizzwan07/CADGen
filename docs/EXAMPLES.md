## Tips for Best Results

- **Always include dimensions** in mm (e.g., "50mm wide" not "wide")
- **Be specific about positions** (e.g., "4 holes spaced 60mm apart")
- **For gears:** specify teeth, module, face width, bore
- **For bolts:** say "with thread grooves" not "with threads"
- **Keep fillets small** relative to the part
- **One change per edit** works best
- **If it fails:** simplify, remove fillets, fewer features
